Start by creating a default iPython profile. From your shell, type:

[sourcecode]$ ipython profile create[/sourcecode]

That should give output similar to this:

[sourcecode][ProfileCreate] Generating default config file: u'/home/mmphego/.ipython/profile_default/ipython_config.py'
[ProfileCreate] Generating default config file: u'/home/mmphego/.ipython/profile_default/ipython_kernel_config.py'[/sourcecode]

Open the newly created file with your favorite text editor, in the above example shown as: [sourcecode]vim /home/mmphego/.ipython/profile_default/ipython_config.py[/sourcecode]

Add the following lines at the bottom:

[sourcecode]c.TerminalInteractiveShell.display_completions = 'readlinelike'[/sourcecode]

Save the file.
